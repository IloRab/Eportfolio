Accueil Expérience Projets
Voici quelques-uns des projets sur lesquels j'ai travaillé pendant mes études en informatiques.

GeoChercheur

(HTML, CSS, JavaScript, PHP, SQL et usage d'API)

Geochercheur est un jeu inspiré de GeoGuessr. Dans le concept de base, le joueur est placé aléatoirement dans un endroit spécifique dans le monde et son objectif est de deviner sa position sur une carte. Nous avons un peu repris l'idée. Mais à la place de le placer aléatoirement, le joueur pourra choisir un certain thème de parcours. Dans ce parcours il y aura plusieurs questions.

Hex

(Java)

Dans ce projet nous avons implémenté le jeu Hex, avec un interface sur la ligne de commande, et deux modes de jeux: "joueur" où deux joueurs s'affrontent, et "seul" où un seul joueur fait face à un ordinateur. Cet ordinateur n'a ici pas d'algorithme complexe, il est très simple grâce à notre architecture de créer un robot plus complexe.

Refonte d'un site web

(HTML, CSS et BootStrap)

Dans le cadre d'un projet visant à refondre le front-end d'un site web appartenant à une agglomération française présentant des défauts majeurs, nous avons décidé de refaire le site web du Beauvaisis.

Démineur

(C++)

Durant ma première année en informatique, dans le cadre de l'apprentissage du C++, il a été demandé de créer un démineur suivant un sujet bien précis. Le sujet consistait à programmer un démineur fonctionnant avec 5 commandes. Chaque commande prend en paramètre son numéro pour effectuer son appel et la taille de la grille de jeu.

Gestion d'un tournois de tennis WTA

(C)

Ce fut le premier projet universitaire en informatique que j'ai fait. Le sujet concernait la gestion d'un tournois de tennis en C. Le sujet comportait plusieurs fonction à implémenter qui permettait la suppression, la création, l'ajout de joueuses dans un classement de tournois.

6 qui prend !

(Java)

Dans ce projet, j'ai implémenté le jeu de carte "6 qui prend !" en Java. Il est possible de jouer Joueur contre Joueur mais aussi Joueur contre Ordinateur. Le projet a était fait dans le cadre de ma première année de BUT informatique.

Demineur

(VB.NET)

Ce projet consiste à la réalisation d’un programme en Visual Basic.NET permettant de jouer une partie du jeu démineur et d’enregistrer les scores des joueurs. Le jeu Démineur étant un jeu de réflexion dont le but est de trouver la localisation de chaque mines cachées sur la grille avec pour seule indication le nombre de mines dans les zones adjacentes.

Les graphes

(Java)

Ce projet consiste à l'implémentation de graphe sur Java. Il est notemment demandé de pouvoir créer leurs matrice ainsi que réaliser deux algorithme sur ces graphes : Dijkstra et Bellman
